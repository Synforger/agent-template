# === sync 対象外の宣言 (= `!` prefix、 理由コメント必須) ===
# src/ に出荷しているが sync では降ろさない file。 init (= src/ 丸ごと rsync) で
# 一度だけ届き、 以降は派生の持ち物になるもの。
# 宣言漏れは `.tooling/local-ci/synced-paths-check.sh` が FAIL で検出する
# (= 「列挙も除外宣言もされていない出荷 file」 = 降ろし忘れの機械検出)。
!CLAUDE.md                  # init で CLAUDE.template.md から展開、 派生で書き換える
!.gitignore                 # init で .gitignore.template から展開、 派生で加筆する
!profile/profile-core.md    # init で profile-core.template.md から展開、 派生で書く
!rules/always.md            # 形態 D: base 出荷は § meta のみ、 派生 personal section と手動 merge
!rules/lazy/_README.md      # 索引の中身は派生固有の lazy 一覧 (= sync すると派生の索引が base の 2 件に潰れる)
!.staledocs.yaml            # pair / ignore が派生固有、 base 版は雛形として参照するもの
